# Code of conduct

vsCodex is committed to a respectful, harassment-free community.

Be constructive, assume good intent, respect different experience levels, and keep technical disagreement focused on the work. Harassment, discrimination, threats, doxxing, sexualized conduct, and repeated personal attacks are not acceptable.

Maintainers may edit or remove contributions and temporarily or permanently restrict participation when conduct is harmful to the project or its community. Enforcement decisions should be proportionate, documented privately, and applied consistently.

Report conduct concerns to the repository owner through their GitHub profile. Do not include sensitive personal information in a public issue. Security vulnerabilities must use the private process in [SECURITY.md](SECURITY.md).
